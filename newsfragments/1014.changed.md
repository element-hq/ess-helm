Set default permissions on Hookshot so that local users only have permissions to use existing integrations.

Permissions should be adjusted to give specific users the ability to administer integrations, e.g.
```yaml
hookshot:
  additional:
    permissions.yaml:
      config: |
        permissions:
        - actor: {{ $.Values.serverName }}
          services:
          - service: "*"
            level: commands
        - action: @an-admin-user:{{ $.Values.serverName }}
          services:
          - service: "*"
            level: admin
```
